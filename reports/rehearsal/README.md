# Rehearsal Archive Artifacts

These are pre-baseline snapshots from `2026-04-13` through `2026-04-19`.
They are kept out of the canonical `data/` tree so `ledger.json` represents
only the official archive lineage that begins on `2026-04-20`.

Full `catalog.json.gz` files are not stored here. They live in the matching
GitHub Release bundles named `rso-archive-YYYY-MM-DD.tar.gz`.
